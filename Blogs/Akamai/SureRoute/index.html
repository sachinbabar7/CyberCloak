Akamai SureRoute

SureRoute is a feature provided by Akamai that helps optimize the delivery of content by selecting the best route for data to travel from the origin server to the end user. It is designed to improve performance and reliability by dynamically choosing the most efficient path for data transmission.

SureRoute works by continuously monitoring the network conditions and performance metrics of various routes between the origin server and the end user. It uses this information to make real-time decisions about which route to use for delivering content. The goal is to minimize latency, reduce packet loss, and ensure a smooth user experience.

By leveraging SureRoute, Akamai can help ensure that content is delivered quickly and reliably, even in the face of network congestion or other issues that may arise along the way. This can be particularly beneficial for websites and applications that require high performance and low latency, such as streaming media, online gaming, and e-commerce platforms.

    Custom Behavior SureRoute setting - Select Origin IP on Child edge server server

    This feature is rarely enabled by itself. In Akamai's advanced metadata configurations, it is usually deployed as a cluster of three parameters to stabilize load-balanced origins:

  • Fixes SureRoute handling of multiple/rotating origin IPs to prevent oscillation and route flapping

    • #on

  • Forces Akamai to use the same origin IP for all requests from a given edge server, even if the origin DNS returns multiple IPs in a round-robin fashion

    • on

  • Disables hostname-specific pinning within the SureRoute statistics key to ensure that performance metrics are aggregated across all origin IPs rather than being split by hostname, which can help with more accurate performance monitoring and troubleshooting

    • off

    By implementing this cluster of parameters, you can help ensure that Akamai's SureRoute feature provides more stable and consistent routing to your load-balanced origins, reducing the likelihood of oscillation and improving overall performance.

    ### ⚠️ Implementation Warning

    Because this directive relies on underlying site-forwarding architecture adjustments, it **cannot be toggled inside Property Manager (Control Center UI)**. You must request Akamai Professional Services or your Solutions Engineer to inject this advanced XML metadata snippet into your property configuration.
